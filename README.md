# site-kurulumu-blm2537 — Üniversite Program Oluşturucu

## Proje Özeti
UniPlan, öğrencilerin haftalık ders programlarını sürükle-bırak yöntemiyle oluşturmasını sağlayan statik bir web sitesidir.

## Sayfalar
- Ana Sayfa (index.html)
- Program Oluşturucu (program.html)
- Galeri (gallery.html)

## Özellikler
- Ortak header + footer
- Harici CSS (css/styles.css) ve JS (js/app.js)
- Program sayfasında ders kartı oluşturma ve tabloya sürükle-bırak ekleme
- Programı Kaydet (localStorage), Temizle, Yazdır/PDF
- Galeride küçük resme tıklayınca modalda büyük görsel açma
